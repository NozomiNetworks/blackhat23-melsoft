# blackhat23-melsoft
Tools to dissect the Mitsubishi Electric MELSOFT protocol, specifically designed to help security teams to investigate network streams independently; multiple PCAPs are also distributed as example to boost the initial learning phase. 

The dissector has also a fully working expert section warning on potential attacks related with the authentication and authorization stages, specifically **CVE-2021-20594**, **CVE-2021-20597**, **CVE-2021-20598**, **CVE-2021-20599**, **NN-2021-0019**.
